# 🃏 Blackjack - Python CLI Game

A Python-based command-line Blackjack game where you play against a computer dealer. The game follows real Blackjack rules and provides a simple, clean interface in the terminal.

---

## 🎮 Features

- ASCII art welcome screen 🎰
- Realistic Blackjack gameplay:
  - Aces count as 1 or 11
  - Blackjack is an automatic win
  - Dealer draws until at least 17
- Hit or Stand options for player
- Repeated gameplay in a loop
- Cross-platform screen clear (Windows/Linux/Mac)
- Clean function-based code
